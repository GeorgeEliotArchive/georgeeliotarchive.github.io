George Eliot Archive API

Simple API endpoint for accessing George Eliot Archive CSV data.

Quick Start

Endpoint: https://georgeeliotarchive.github.io/api/csv.html

Live Examples

  • Get all items (CSV)
  • Search for "1875" (CSV)
  • Search for "1875" (JSON)
  • Search for "reading" (JSON)

Parameters

Parameter Type Description Default
searchTerm string Search term to filter results empty (returns all)
format string Response format: csv or json csv
callback string JSONP callback function name none

Example Usage

JavaScript Fetch

fetch('https://georgeeliotarchive.github.io/api/csv.html?searchTerm=1875')
  .then(response => response.text())
  .then(csvData => {
    console.log(csvData);
  });

With JSON Format

fetch('https://georgeeliotarchive.github.io/api/csv.html?searchTerm=1875&format=json')
  .then(response => response.json())
  .then(data => {
    if (data.success) {
      console.log(data.data); // CSV content
    }
  });
